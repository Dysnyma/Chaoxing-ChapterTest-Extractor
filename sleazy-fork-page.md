# 学习通章节测验提取器

> 自动提取学习通课程所有章节测验的题目和答案，支持字体解密、多格式导出、AI智能解析。

---

## 功能特性

- **自动发现章节测验** — 在课程主页一键扫描所有章节测验，无需手动逐个进入
- **字体解密** — 内置 Canvas 字形对比法，自动解密学习通的 font-cxsecret 加密字体
- **当前页提取** — 在任意测验页面直接提取当前页题目，灵活方便
- **多格式导出** —
  - **TXT** — 纯文本，含完整题目、选项、答案、解析
  - **JSON** — 结构化数据，便于二次处理
  - **Markdown** — 排版清晰，适合阅读和笔记
  - **XLSX** — 标准 Excel 格式，列包括：试题类型、试题题目、参考答案、各选项、章节、解析
- **AI 智能解析** — 支持 OpenAI 兼容 API（DeepSeek 等），自动为题目生成答案解析
- **可配置关键词** — 自定义匹配测验的关键词，适应不同课程的命名习惯
- **可拖拽面板** — 控制面板可自由拖动，不遮挡页面内容

## 支持的题型

| 题型 | 说明 |
|------|------|
| 单选题 | 自动提取正确选项字母 |
| 多选题 | 自动提取所有正确选项 |
| 判断题 | 自动转换为"正确"/"错误" |
| 填空题 | 提取标准答案 |
| 简答题 | 提取参考答案 |

## 使用方法

### 自动提取全部（推荐）

1. 打开课程主页（`mooc2-ans.chaoxing.com/mooc2-ans/mycourse/stu?......`）
2. 等待页面完全加载（左侧课程目录出现）
3. 点击右侧面板的 **「▶ 自动提取全部」**
4. 脚本自动扫描所有章节测验，逐个加载并提取
5. 提取完成后，点击导出按钮保存

### 提取当前页

1. 进入任意章节测验页面（能看到题目和答案的页面）
2. 点击 **「📄 提取当前页」**
3. 可多次在不同页面提取，结果会累积
4. 最后统一导出

### AI 生成解析

1. 点击 **「🤖 AI解析」** 展开设置
2. 填写 API 地址、Key 和模型名（支持 DeepSeek、OpenAI 等兼容接口）
3. 提取题目后，点击 **「🤖 开始生成解析」**
4. 生成完成后重新导出文件即可

### 自定义匹配关键词

如果某些测验未被识别，可点击 **「⚙️ 关键词」** 添加课程特定的测验名称关键词。

## 支持的页面

| 页面 | URL 匹配 |
|------|----------|
| 课程主页 | `mooc2-ans.chaoxing.com/mooc2-ans/mycourse/stu*` |
| 知识卡片 | `mooc1.chaoxing.com/mooc-ans/knowledge/cards*` |

## 注意事项

1. **需要先完成测验** — 只有提交并查看答案后，答案才会显示在页面上
2. **在课程主页运行** — 「自动提取全部」功能需要在课程主页（能看到完整目录的页面）使用
3. **提取需要时间** — 每个章节有 2-4 秒的间隔，避免请求过快
4. **保持页面打开** — 提取过程中请勿关闭或刷新页面
5. **字体解密依赖页面元素** — 确保页面中使用了加密字体的元素已渲染

## 技术原理

### 字体解密

部分学习通课程的题目使用 `font-cxsecret` 自定义字体进行混淆。脚本通过比对 Canvas 渲染的字形像素特征，将混淆字符映射回真实字符。

### 章节发现（多重策略）

1. 从 `studentcourse` iframe 读取 DOM 结构
2. HTTP 请求获取课程结构页面并解析
3. 扫描主页面中的章节容器元素
4. 解析页面脚本中的 JSON 数据
5. 通过 transfer API 获取课程结构

### 题目提取

- 优先从页面的 `mArg` JavaScript 变量中解析结构化数据
- 回退方案：从 DOM 元素（`.TiMu`）中提取题目信息
- 自动识别题型、选项、答案和解析

### TXT 格式

```
══════════════════════════════════════════════════
  课程：高等数学（上）
  时间：2026/6/25 15:30:00
  共 3 个章节，25 道题目
══════════════════════════════════════════════════

┌────────────────────────────────────────────────┐
│ 第一章 函数与极限 - 章节测验（10题）
└────────────────────────────────────────────────┘

  1. [单选] 函数 f(x)=1/x 的定义域是？
       A. (-∞, 0) ∪ (0, +∞)
       B. (-∞, +∞)
       C. [0, +∞)
       D. (0, +∞)

  ▶ 答案：A
  ✎ 解析：分母不能为零，故 x≠0
```
